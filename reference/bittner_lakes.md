# Bittner Creek lakes

FWA lake polygons within the Bittner Creek watershed.

## Usage

``` r
bittner_lakes
```

## Format

An `sf` data frame with columns:

- name:

  Lake name from BC geographic names

- area_ha:

  Lake area in hectares

- geom:

  Polygon geometry (WGS 84)

## Source

BC Freshwater Atlas via newgraph database
