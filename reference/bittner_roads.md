# Bittner Creek roads

DRA road segments within expanded Bittner Creek watershed.

## Usage

``` r
bittner_roads
```

## Format

An `sf` data frame with columns:

- road_type:

  Transport line type code (RH1, RA1, RA2, etc.)

- geom:

  Linestring geometry (WGS 84)

## Source

BC Digital Road Atlas via newgraph database
