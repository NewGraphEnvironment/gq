# Bittner Creek stream network

FWA stream segments within the Bittner Creek watershed (order \>= 2).

## Usage

``` r
bittner_streams
```

## Format

An `sf` data frame with columns:

- gnis_name:

  Stream name from BC geographic names

- stream_order:

  Strahler stream order

- geom:

  Linestring geometry (WGS 84)

## Source

BC Freshwater Atlas via newgraph database
