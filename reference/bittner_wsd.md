# Bittner Creek watershed boundary

Watershed polygon for Bittner Creek near Prince George, BC (~56 km²).
Pulled from the BC Freshwater Atlas via fwapg.

## Usage

``` r
bittner_wsd
```

## Format

An `sf` data frame with 1 row and columns:

- area_ha:

  Watershed area in hectares

- geom:

  Polygon geometry (WGS 84)

## Source

BC Freshwater Atlas via newgraph database
