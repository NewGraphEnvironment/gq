# Neexdzii Kwa crossings

Stream crossings from bcfishpass within the Neexdzii Kwa subbasin.

## Usage

``` r
neexdzii_crossings
```

## Format

An `sf` data frame with columns including:

- barrier_status:

  Barrier status (BARRIER, PASSABLE, POTENTIAL, UNKNOWN)

- pscis_status:

  PSCIS assessment status

- crossing_source:

  Source of crossing data

- geom:

  Point geometry (WGS 84)

## Source

bcfishpass via newgraph database and fresh package
