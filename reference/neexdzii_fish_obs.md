# Neexdzii Kwa fish observations

Fish observation events from FISS within the Neexdzii Kwa subbasin.

## Usage

``` r
neexdzii_fish_obs
```

## Format

An `sf` data frame with columns including:

- species_code:

  Species code (CO, CH, RB, etc.)

- geom:

  Point geometry (WGS 84)

## Source

bcfishobs via newgraph database and fresh package
