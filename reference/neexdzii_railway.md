# Neexdzii Kwa railway

Railway track segments clipped to the Neexdzii Kwa subbasin.

## Usage

``` r
neexdzii_railway
```

## Format

An `sf` data frame with columns:

- track_name:

  Railway track name

- geom:

  Linestring geometry (WGS 84)

## Source

BC GBA Railway Tracks via newgraph database
