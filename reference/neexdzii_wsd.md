# Neexdzii Kwa subbasin watershed boundary

Subbasin polygon on the Neexdzii Kwa (Upper Bulkley River) between
Johnny David Creek (downstream) and Richfield Creek (upstream) in the
traditional territory of the Wet'suwet'en. ~212 km². Generated by
watershed subtraction using fresh.

## Usage

``` r
neexdzii_wsd
```

## Format

An `sf` data frame with 1 row and polygon geometry (WGS 84).

## Source

BC Freshwater Atlas via newgraph database and fresh package
