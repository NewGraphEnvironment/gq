# Bulkley and Morice watershed groups

Watershed group polygons for BULK and MORR for keymap inset context.

## Usage

``` r
neexdzii_wsg
```

## Format

An `sf` data frame with columns:

- watershed_group_code:

  FWA watershed group code (BULK, MORR)

- geom:

  Polygon geometry (WGS 84)

## Source

BC Freshwater Atlas via newgraph database
